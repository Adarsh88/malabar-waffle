# Malabar Waffle Angular Project

Full standalone Angular project scaffold with:
- Standalone components and route-level lazy loading
- Signals-based state examples
- SSR-ready configuration
- PWA manifest and service worker config
- SEO-friendly route titles
- Reusable feature composition

## Run

```bash
npm install
npm start
```

## Production build

```bash
npm run build
```

## SSR serve

```bash
npm run serve:ssr
```
